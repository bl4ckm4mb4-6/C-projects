# C Person Library (Opaque Struct & Encapsulation)

A didactic example of how to implement the concept of **Encapsulation** and **Information Hiding** in standard C.



## 🧠 Key concepts covered

* **Opaque Structs:** Definition of the data structure hidden from the user in the `.c` file.

* **Dynamic Allocation:** Safe Heap memory management using `malloc` and `free`.

* **Public vs Private Interface:** Use of Get/Set functions to manipulate data.



## 🚀 How to compile and run (GCC)
In Dev-C++, you compile the file first and then run it.

In the command prompt or PowerShell, use this commands:

```bash
    gcc main.c persona.c -o programma
    
    ./programma
```
